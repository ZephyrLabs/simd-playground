# SIMD library with AVX2

This library has various prototypes and function implmentations for doing vector operations 
using the AVX2 intrinsics in the **immintrin.h** library.

Function list:

* Vector Arithmetic:
    - add
    - subtract
    - multiply
    - divide
    - square root

* Matrix Artithmetic:
    - add
    - subtract
    - multiply

* Matrix Manipulation:
    - transposing
    - inverse
