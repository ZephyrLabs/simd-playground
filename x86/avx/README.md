# SIMD library with AVX

This library has various prototypes and function implmentations for doing vector operations 
using the AVX/AVX2 intrinsics in the **immintrin.h** library.

Function list:

* Vector Arithmetic:
    - add
    - subtract
    - multiply
    - divide

* Tensor Artithmetic:
    - add
    - subtract
    - multiply

* Tensor Manipulation:
    - transposing
    - inverse
